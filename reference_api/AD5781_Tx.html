adi.AD5781.Tx#

AD5781 Voltage output DAC Class

adi.AD5781.Tx Transmits data to the AD5781 DAC The adi.AD5781.Tx System object is a signal sink that can transmit data to the AD5781.

tx = adi.AD5781.Tx; tx = adi.AD5781.Tx(‘uri’,’ip:192.168.2.1’);
