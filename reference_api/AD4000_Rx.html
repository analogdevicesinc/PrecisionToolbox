adi.AD4000.Rx

AD4000 Precision ADC Class

adi.AD4000.Rx Receives data from the AD4000 ADC The adi.AD4000.Rx System object is a signal source that can receive data from the AD4000.

rx = adi.AD4000.Rx; rx = adi.AD4000.Rx(‘uri’,’192.168.2.1’);
