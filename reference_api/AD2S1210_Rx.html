adi.AD2S1210.Rx#

AD2S1210 Resolver-to-Digital Converter Class

adi.AD2S1210.Rx Receives data from the AD2S1210 Resolver The adi.AD2S1210.Rx System object is a signal source that can receive data from the AD2S1210.

rx = adi.AD2S1210.Rx; rx = adi.AD2S1210.Rx(‘uri’,’192.168.2.1’);
