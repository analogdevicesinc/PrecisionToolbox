adi.AD4007.Rx¶

AD4007 Precision ADC Class

adi.AD4007.Rx Receives data from the AD4007 ADC The adi.AD4007.Rx System object is a signal source that can receive data from the AD4007.

rx = adi.AD4007.Rx; rx = adi.AD4007.Rx(‘uri’,’192.168.2.1’);

AD4007 Datasheet
