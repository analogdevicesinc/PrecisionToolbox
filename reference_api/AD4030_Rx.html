adi.AD4030.Rx¶

AD4030-24 Precision ADC Class

adi.AD4030.Rx Receives data from the AD4030-24 ADC The adi.AD4030.Rx System object is a signal source that can receive data from the AD4030-24.

rx = adi.AD4030.Rx; rx = adi.AD4030.Rx(‘uri’,’192.168.2.1’);

AD4030-24 Datasheet
