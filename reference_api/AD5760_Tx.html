adi.AD5760.Tx¶

AD5760 Voltage output DAC Class

adi.AD5760.Tx Transmits data to the AD5760 DAC The adi.AD5760.Tx System object is a signal sink that can transmit data to the AD5760.

tx = adi.AD5760.Tx; tx = adi.AD5760.Tx(‘uri’,’ip:192.168.2.1’);

AD5760 Datasheet
