adi.AD4021.Rx¶

AD4021 Precision ADC Class

adi.AD4021.Rx Receives data from the AD4021 ADC The adi.AD4021.Rx System object is a signal source that can receive data from the AD4021.

rx = adi.AD4021.Rx; rx = adi.AD4021.Rx(‘uri’,’192.168.2.1’);
