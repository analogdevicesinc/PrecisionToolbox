adi.AD7124_8.Rx

AD7124-8 Precision ADC Class

adi.AD7124-8.Rx Receives data from the AD7124 ADC The adi.AD7124-8.Rx System object is a signal source that can receive data from the AD7124-8.

rx = adi.AD7124_8.Rx; rx = adi.AD7124_8.Rx(‘uri’,’ip:192.168.2.1’);
