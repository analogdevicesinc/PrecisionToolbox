adi.AD4004.Rx#

AD4004 Precision ADC Class

adi.AD4004.Rx Receives data from the AD4004 ADC The adi.AD4004.Rx System object is a signal source that can receive data from the AD4004.

rx = adi.AD4004.Rx; rx = adi.AD4004.Rx(‘uri’,’192.168.2.1’);
